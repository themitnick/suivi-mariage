Questions fréquentes

Tout ce que vous devez savoir sur notre service

@if (openFaq === 0) {

Pour effectuer votre réservation, vous aurez besoin des pièces d'identité des deux futurs époux (carte nationale d'identité ou passeport), des extraits de naissance datant de moins de 3 mois, et éventuellement des documents de divorce ou de veuvage si applicable.

}
@if (openFaq === 1) {

Nous recommandons de faire votre réservation au moins 2 semaines à l'avance pour avoir plus de choix dans les créneaux disponibles. Cependant, il est possible de réserver jusqu'à 3 jours avant la date souhaitée, sous réserve de disponibilité.

}
@if (openFaq === 2) {

Une fois votre réservation validée, vous recevrez un code de confirmation par SMS et email. Vous pourrez utiliser ce code pour accéder à votre espace personnel et communiquer avec votre célébrant. Celui-ci vous contactera pour finaliser les détails de votre cérémonie.

}
@if (openFaq === 3) {

Oui, vous pouvez modifier votre réservation jusqu'à 5 jours avant la date prévue, sous réserve de disponibilité. Pour les annulations, elles sont possibles jusqu'à 48h avant la cérémonie. Utilisez votre espace personnel ou contactez directement votre célébrant.

}
@if (openFaq === 4) {

Oui, notre plateforme de réservation en ligne est entièrement gratuite. Seuls les frais officiels de mariage civil fixés par la mairie s'appliquent (ces frais sont les mêmes que pour une démarche traditionnelle).

}
